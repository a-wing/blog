git提取出两个版本之间的差异文件并打包

Sep 4, 2016 • shiyongxin

sudo apt-get install zip

就能成功打包了。只是这样的话也太麻烦了吧，幸好linux有提供一个命令xargs能将前一个命令的输出转成另外一个命令的参数，按照这个思路就非常简单了。

$ git diff 608e120 4abe32e --name-only | xargs zip update.zip
